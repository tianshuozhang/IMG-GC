Welcome to my website

This website is dedicated to providing easy-to-use visual tools to help beginners without coding experience to experience AI generation.

Here, we offer two technologies: text-to-image and image-to-image generation, which can be accessed through the following links.

img2text
img2img

Description

Img to Text

One can obtain an image by providing a URL or uploading a file. Users have the option to ask a question or provide a prompt in order to receive the desired content. Additionally, the prompt field can be left empty if desired.

We use the pre-trained model BLIP from Hugging Face, which can be accessed through the link : Salesforce/blip2-opt-2.7b

Img to Img

By providing a URL or uploading a file, users can obtain an image. They can then use prompts to generate the desired image. However, please make sure not to leave the prompt field empty

We use the pre-trained model stable-diffusion from Hugging Face, which can be accessed through the link : stabilityai/stable-diffusion-2
